## 技术实现细节

### 后端API (运行在 localhost:8001)
1. **任务分解API**: `POST /api/ai/task-decomposition`
   - 接收项目信息，模拟AI处理
   - 创建项目并生成优先任务
   - 存储任务到数据库

2. **任务过滤API**: `GET /api/tasks/filter/{period}`
   - 支持 day/week/month 过滤
   - 返回按时间过滤的任务列表

3. **甘特图API**: `GET /api/gantt/{project_id}`
   - 返回项目甘特图数据
   - 包含阶段和任务信息
   - 支持时间轴显示

### 前端功能
1. **任务分解模块**:
   - 调用真实AI API
   - 显示分解结果
   - 提供导航到优先级管理

2. **优先级管理模块**:
   - 显示任务列表
   - 支持时间过滤
   - 任务拖拽调整（待实现）

3. **可视化Dashboard**:
   - 新增"时间线"标签页
   - 甘特图显示任务时间轴
   - 阶段可展开/收起
   - 支持缩放和重置
